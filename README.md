Данный репозиторий содержит аналитический проект, выполненный для анализа экономики игры «Секреты Темнолесья». Основная цель — исследование поведения игроков и их покупательской активности при совершении внутриигровых покупок за виртуальную валюту «райские лепестки».
Цель проекта: Изучить влияние характеристик игроков и их игровых персонажей (в частности, расы) на покупку внутриигровой валюты, а также оценить активность игроков при совершении транзакций для формирования стратегических рекомендаций по монетизации.
