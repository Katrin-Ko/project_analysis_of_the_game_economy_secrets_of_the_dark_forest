В этом репозитории представлен аналитический проект, выполненный для анализа экономики мобильной игры «Секреты Темнолесья». В рамках проекта было проведено исследование поведения игроков и их покупательской активности при совершении внутриигровых покупок за виртуальную валюту «райские лепестки».

Цель проекта: изучить влияние характеристик игроков и их игровых персонажей на покупку внутриигровой валюты, а также оценить активность игроков при совершении транзакций для формирования стратегических рекомендаций по монетизации.

Основные задачи:
1. Исследовательский анализ данных: определить долю платящих игроков, проанализировать структуру покупок, выявить аномальные транзакции и популярные игровые предметы.
2. Анализ зависимости активности от расы персонажа: выявить, существует ли связь между выбором расы персонажа и покупательским поведением игроков, включая частоту, стоимость и общую сумму покупок.
3. Сравнительный анализ: определить ключевые различия в поведении игроков разных рас (Human, Elf, Demon, Northman, Orc, Angel) по метрикам монетизации.
4. Разработка рекомендаций: сформулировать конкретные предложения по оптимизации системы монетизации, балансировке игрового процесса и развитию внутриигрового магазина с учетом выявленной закономерности поведения игроков.
